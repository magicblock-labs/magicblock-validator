# Agent Knowledge Base

This directory is the repository-local knowledge base for AI coding agents.

- `rules/` contains invariant behavior and decision rules.
- `context/` contains static reference material such as overview, architecture, crate map, and crate-specific guides.
- `memory/` contains durable project-memory and documentation-stewardship rules.
- `specs/` contains protocol and feature specifications.
- `skills/` contains executable scripts or capabilities agents can run.
- `personas/` contains specialized agent profiles when needed.

Start with `../AGENTS.md`, then read the relevant files for the change being made.
